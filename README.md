# EmporiUm Power BI Sales Dashboard

## Overview
This project is an analysis of EmporiUm’s sales performance. The goal is to evaluate sales trends, understand regional performance, and present insights using interactive visualizations.

## Objectives
- Prepare and clean the provided datasets using Power Query.
- Build a functional data model with correct relationships.
- Create DAX measures for key performance indicators (KPIs).
- Develop a multi-page Power BI report with interactive visuals.
- Analyze sales performance for the Northeast region.
- Present findings through a structured dashboard layout.

## Tools Used
- Power BI Desktop  
- Power Query  
- DAX 


## Data Sources
The project uses the following datasets provided:

- Store Sales  
- Online Sales  
- Store Location  
- Product Information  


All data cleaning and transformations were performed in Power Query.
